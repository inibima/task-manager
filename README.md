## Instalasi Manual
1. `composer install`  
2. `cp .env.example .env` + atur DB  
3. `php artisan key:generate`  
4. `php artisan migrate`  
5. `php artisan serve`

## Akses
- **Admin**: full akses `/admin`  
- **Manager**: lihat laporan  
- **Staff**: update status tugas

## Dokumentasi
